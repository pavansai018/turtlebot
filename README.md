
## SLAM Mapping Demo

This video demonstrates the robot operating in **Mapping Mode (SLAM)** inside the custom maze world.

### What the Video Shows

- Gazebo simulation running with the maze environment  
- TurtleBot exploring the environment  
- Live LiDAR scans being visualized in RViz  
- Real-time occupancy grid map generation  
- Gradual map refinement as the robot explores new regions
